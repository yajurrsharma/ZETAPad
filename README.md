# ZETAPad


## TLDR; Simple Description
A simple macropad (or fancy word, *hackpad*) made using a Seeed Studio XIAO RP2040. Named after the Greek symbol Zeta (also the small letter symbol for Zeta resembles a Y, which is the first letter of my name!)

## Parts
1) 0.91" inch OLED Display (SSD1306)
2) Seeed Studio XIAO RP2040
3) 4 Cherry MX Keys
4) 1 Rotary Encoder Switch
